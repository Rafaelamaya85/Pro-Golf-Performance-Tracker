# The Heat Index

The Heat Index is a golf rankings website that highlights the top 5 hottest players on the PGA Tour and LPGA Tour based on current-season performance. Instead of using traditional world rankings, this project uses a simpler scoring formula based on wins, top 10 finishes, majors won, and cuts missed to show which golfers are performing the best right now.

## Project Purpose
The purpose of this project is to create a simpler and more current way to rank golfers based on their performance this season. As a golf fan, I wanted to build a website that makes rankings easier to understand and more focused on who is actually playing the best right now.

## Features
- Homepage with current rankings
- About page explaining the ranking system
- Past Rankings page to track weekly changes
- PGA and LPGA player content
- Real player images and golf-related content
- Navigation between pages
