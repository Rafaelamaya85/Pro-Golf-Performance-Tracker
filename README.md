# The Heat Index

The Heat Index is a golf rankings website that highlights the top 5 hottest players on the PGA Tour and LPGA Tour based on current-season performance. Instead of using traditional world rankings, this project uses a simpler scoring formula based on wins, top 10 finishes, majors won, and cuts missed to show which golfers are performing the best right now.

## Project Purpose
The purpose of this project is to create a simpler and more current way to rank golfers based on their performance this season. As a golf fan, I wanted to build a website that makes rankings easier to understand and more focused on who is actually playing the best right now. The previous rankings employed by both the PGA/LPGA tours used complex systems based on 2–3 years of results, meaning a player could be ranked at the top without currently playing well. This site uses only current season data to make it easier to identify who is truly playing the best right now.

## What Changed from P01 to P02

From Project 01 to Project 02, the site evolved significantly through iteration, testing, and feedback.

The biggest improvements were in both design and usability. The homepage was upgraded with a more polished layout, including a hero section, stat cards, and improved leaderboard design with better spacing and alignment. I also added visual enhancements such as hover effects, movement indicators for rankings, and stronger visual hierarchy to make the site easier to scan and understand.

Functionality also improved. I added filtering between PGA and LPGA players, clickable player profile pages, and updated ranking data to reflect real tournament results more consistently. The Past Rankings page was expanded to better track weekly changes and provide more context over time.

Another important improvement was responsiveness. I made updates to ensure the site works better across different screen sizes, especially on mobile devices, since the goal is for users to quickly check rankings from anywhere.

Overall, these changes were guided by peer feedback, personal testing, and a focus on making the site clearer, more useful, and more visually polished.
